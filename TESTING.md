## Регресійний чек-лист UI (після кожного блоку з Блоку 3+)

Після будь-яких змін у компонентах пройтися по всіх 4 основних сторінках:

| Сторінка | Перевірити |
|---|---|
| `/` (Dashboard) | Метрики, таблиця «Poslední nabídky», Aktivita — дані є, вигляд не змінився |
| `/nabidky` | Список КП, таби-фільтри, статус-беджі — кольори правильні |
| `/katalog` | Грід продуктів, пошук, фільтр по типу — відпрацьовує |
| `/nabidky/:id` | Таблиця позицій, підсумок, кнопки дій — жодних зсувів layout |
| `/nabidka/:shareId` | Публічна сторінка без сайдбару — брендинг, таблиця, форма коментаря |

- Акцентний колір змінився (має бути `#8B7355`)
- З'явились фіолетові/неонові елементи
- Числа відображаються не Geist Mono
- Emoji в UI
- Fontshare CDN запит у Network
